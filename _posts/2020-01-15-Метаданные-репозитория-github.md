# »спользование

Repository metadata is exposed to your Jekyll siteТs configuration in the `site.github` namespace.

Simply, reference any of the below keys as you would any other site configuration value present in your `_config.yml` file, prefacing the key with `site.github`.

For example, to list a projectТs name, you might write something like The project is called github-metadata or to list an organizationТs open source repositories, you might use the following:

```
{% for repository in site.github.public_repositories %}
  * [{{ repository.name }}]({{ repository.html_url }})
{% endfor %}
```
